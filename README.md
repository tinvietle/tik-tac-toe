# Tik-tac-toe through the use of fully AI-generated code
The project uses structured markdown instructions to guide the LLM to do the project. This is part of a subject at school. The purpose is to exploit the power of AI and its limitations.
To construct the prompt, learn Prompt Engineeer template given by Anthropic.
LLM Model: GPT-5.3-Codex
