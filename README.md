# Tic Tac Toe with AI-Generated Code

This project explores building Tic Tac Toe using structured Markdown instructions to guide LLMs.

It is part of a school subject, with the goal of understanding both the power and limitations of AI-assisted development.

## Stack and Tools

- Models: GPT-5.3-Codex, Claude Sonnet 4.6
- Editor: VS Code chat workflow
- Prompting approach: Anthropic Prompt Engineer template

## Project Timeline

### Week 1

Java CLI Tic Tac Toe.

Process:
- Prepare prompt
- Send prompt to model
- Wait for completion
- Fully vibe coding in around 5 minutes

Run:

```bash
cd week1
java -cp target/tik-tac-toe-1.0.0.jar com.progex.tictactoe.Main 1
```
